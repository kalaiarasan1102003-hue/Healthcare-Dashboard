🏥 Healthcare Analytics Dashboard

 🎯 Project Objective
To analyze healthcare data and generate insights related to patient trends, hospital performance, cost analysis, and treatment efficiency.

---

 🧠 Business Problem
Healthcare organizations face challenges such as:
- High treatment costs
- Inefficient hospital resource usage
- Lack of patient trend analysis
- Difficulty in performance tracking

This dashboard helps solve these problems using data visualization.

---

 🛠 Tools Used
- SQL
- Power BI
- Excel

---

 📂 Dataset Information
The dataset includes:
- Patient details
- Treatment records
- Hospital information
- Cost and billing data
- Diagnosis categories

---

 📊 Key Metrics
- Total patients treated
- Average treatment cost
- Hospital performance comparison
- Disease category distribution
- Admission vs discharge trends

---

 📈 Key Insights
- Identified high-cost treatment categories
- Found busiest hospitals with maximum patients
- Analyzed seasonal admission trends
- Compared hospital efficiency based on patient outcomes
